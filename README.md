# Email/SMS Spam Classifier

This project is a simple Email/SMS Spam Classifier built using natural language processing (NLP) techniques to classify messages as spam or not spam (ham).

## Overview

Email and SMS spam are common annoyances for users. This classifier helps in identifying whether a given message is likely to be spam or not. It leverages machine learning algorithms and NLP preprocessing to make predictions.

## Features

- Classifies messages into two categories: spam and not spam (ham).
- Provides a user-friendly interface for inputting messages and obtaining predictions.
- Utilizes NLTK for text preprocessing and a pre-trained model for classification.

## Technologies Used

- Python
- Streamlit
- NLTK (Natural Language Toolkit)
- Scikit-learn
